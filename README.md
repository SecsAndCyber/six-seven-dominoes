# six-seven-dominoes
Lightweight, mobile dominoes game

## Mobile export

This project includes helper files for exporting to Android and iOS.

- Android export preset: `export_presets.cfg` (Android section)
- iOS export preset: `export_presets.cfg` (iOS section)
- Android helper: `android/AndroidManifest.tmpl` and `android/README-keystore.md`
- iOS helper: `ios/README.md`

See those files for step-by-step export instructions and placeholders you must fill (package identifiers, keystore paths, signing, and provisioning).
